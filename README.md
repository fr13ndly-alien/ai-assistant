# AI Assistant Playground

Playground repository for an AI assistant chatbot that communicates through Telegram and can be customized with Goclaw-powered features.

## Getting Started

1. Copy `.env.example` to `.env`.
2. Fill in the required Telegram and assistant configuration values.
3. Add application code under this repository as the playground evolves.

## Configuration

Keep secrets in `.env`; it is intentionally ignored by Git.
